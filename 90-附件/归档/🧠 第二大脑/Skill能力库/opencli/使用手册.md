## 命令分类

### 🔓 公共 API 命令（无需浏览器）

这些命令直接调用网站的公开 API，**在 WorkBuddy 内可直接使用**：

| 命令 | 功能 | 示例 |
|------|------|------|
| `hackernews top` | Hacker News 热榜 | `opencli hackernews top --limit 10` |
| `hackernews new` | Hacker News 最新 | `opencli hackernews new --limit 5` |
| `v2ex hot` | V2EX 热门 | `opencli v2ex hot --limit 10` |
| `v2ex latest` | V2EX 最新 | `opencli v2ex latest` |
| `github search` | GitHub 搜索 | `opencli github search --query "AI"` |
| `bbc news` | BBC 新闻 | `opencli bbc news --limit 5` |

**输出格式选项：**
```bash
-f table   # 表格（默认）
-f json    # JSON
-f md      # Markdown
-f csv     # CSV
```

## 使用场景

### 在 WorkBuddy 内使用（公共 API）

适合：新闻获取、技术趋势、日常信息收集

```
用户: "看看 Hacker News 热榜"
AI: 执行 opencli hackernews top --limit 10
```

### 在本地终端使用（浏览器命令）

适合：需要登录态的个性化内容

```powershell
# 查看 B 站动态
opencli bilibili feed --limit 10

# 搜索知乎
opencli zhihu search --query "AI 编程"

# 查看雪球热门股票
opencli xueqiu hot-stock
```

## 故障排除

| 问题 | 解决 |
|------|------|
| `Playwright MCP server not found` | 先启动 MCP 服务器（见步骤 3） |
| 获取空数据 | 检查 Chrome 是否已登录目标网站 |
| 中文乱码 | PowerShell 使用 UTF-8: `[Console]::OutputEncoding = [System.Text.Encoding]::UTF8` |
