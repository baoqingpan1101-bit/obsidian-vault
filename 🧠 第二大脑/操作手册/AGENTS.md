# 🤖 AGENTS — 标准操作流程 (SOP)

> 操作手册层 · 中频更新 · 定义 AI 在各场景下的标准行为

---

## 🚀 启动流程（每次新会话）

```
1. 读取 🧠 第二大脑/L3-核心记忆/USER.md     → 了解你是谁、当前关注点
2. 读取 🧠 第二大脑/操作手册/TOOLS.md        → 了解当前环境配置
3. 检查 🧠 第二大脑/任务看板/🔴-阻塞中.md   → 有没有上次未解决的阻塞
4. 检查 01-收件箱/ 是否有待处理内容
5. 回报状态："已就绪，当前阻塞 X 项，收件箱 X 条"
```

---

## 📋 任务处理流程

### 简单任务（≤2步）
```
接收 → 执行 → 回报结果
```

### 复杂任务（≥3步）
```
接收 → 列 Todo 清单 → 逐项执行 → 完成后写入工作日志
```

### 阻塞处理
```
发现阻塞 → 写入 🔴-阻塞中.md（注明原因+需要什么）→ 告知你 → 等待授权
```

---

## 📔 日记记录 SOP

触发词：「帮我写今天日记」/「记录今天...」

```
1. 确认日期（默认当天）
2. 写入路径：20-日志/日记/YYYY/MM/YYYY-MM-DD.md
3. 使用简洁模板（3块：今天 / 感悟 / 消费）
4. 如有消费数据，同步写入当月账单
5. 更新 HOME.md 的「最近日记」区块（如有静态链接）
```

---

## 💰 账单记录 SOP

触发词：「记账」/「花了」/「买了」

```
1. 解析：日期 + 项目 + 类别 + 金额
2. 写入路径：03-领域/财务规划/记账/YYYY-MM 账单.md
3. 追加到对应日期分组
4. 更新月度汇总（餐饮/购物/交通...各分类小计）
```

---

## 📅 周记撰写 SOP

触发词：「帮我写本周周记」/「写周记」

```
1. 读取本周（周一到今天）所有日记文件
2. 提炼：工作进展 / 关键事件 / 本周消费汇总 / 感悟
3. 写入路径：20-日志/周记/YYYY-Wxx.md
4. 关联本周每日日记（wikilink）
```

---

## 🔍 技能安装 SOP

触发词：「安装 Skill」/「装...技能」

```
1. 获取 SKILL.md 原始内容
2. 执行安全审计（参考 SOUL.md 红线）
3. 输出审计报告：P0/P1/P2 定级
4. P2 才可安装，P0/P1 需明确授权
5. 安装后记录到：🧠 第二大脑/Skill能力库/README.md
6. 写入今日工作日志
```

---

## 🔄 月度系统复盘 SOP

触发词：「帮我做本月复盘」/ HEARTBEAT 每月末触发

```
1. 读取本月所有工作日志（🧠 第二大脑/工作日志/YYYY/YYYY-MM-*.md）
2. 统计：任务数、阻塞数、Skill 变更数
3. 识别反复出现的问题 → 提炼到专题知识库/踩坑记录/
4. 评估系统运转情况 → 写入复盘报告
5. 主动提出 2-3 条改进建议 → 等待你确认是否实施
6. 写入路径：🧠 第二大脑/工作日志/YYYY/YYYY-MM-月度复盘.md
```

---

## 🛠️ 报错处理流程

```
1. 截图/记录完整错误信息
2. 在 vault 中搜索是否有相同错误的历史记录（L1工作日志）
3. 有历史 → 按记录方案处理
4. 无历史 → 分析原因，尝试修复，记录到今日日志
5. 无法解决 → 标记为阻塞，写入 🔴-阻塞中.md
```
