# Life Knowledge System

基于 **PARA + MOC + Zettelkasten** 的个人知识管理系统，使用 Obsidian 构建。

## 快速上手

1. 用 Obsidian 打开本目录（「打开文件夹作为库」）
2. 信任并启用插件
3. 打开 `00-系统/📖 使用手册.md` 查看完整说明

## 目录结构

```
├── 00-系统/        # 模板、手册
│   └── templates/  # 笔记模板
├── 01-收件箱/      # 临时捕获区（每日清理）
├── 02-知识库/      # 原料笔记（剪藏提炼后）
│   ├── 专业/       # 工程专业知识
│   ├── 计算机/     # 技术、编程
│   ├── 管理/       # 项目管理、方法论
│   ├── 个人成长/   # 学习、自我提升
│   └── 杂项/       # 待归类内容
├── 03-读书/        # 读书笔记
├── 04-记账/        # 月度账单
├── 05-剪藏/        # 原文存档
├── 06-日记/        # 每日记录
│   └── 2026/       # 按年份分组
├── 07-卡片盒/      # Zettelkasten 永久笔记
│   ├── 概念卡片/   # 定义、解释
│   ├── 方法卡片/   # 可复用方法论
│   ├── 观点卡片/   # 个人洞察
│   └── 索引卡片/   # MOC 知识地图
├── 🧠 AI大脑/      # AI 代理系统
│   ├── USER.md     # 用户画像
│   ├── AGENTS.md   # 操作手册
│   └── 工作日志/   # 操作记录
└── 90-附件/        # 图片、文件、归档
```
